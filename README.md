# 交互式 3D 塔罗牌

一个基于手势控制的 3D 塔罗牌阅读体验。

## 功能特性
-   **3D 牌阵**: 78 张韦特塔罗牌（Rider-Waite）排列成沉浸式的 3D 圆柱形阵列。
-   **手势追踪**: 通过 MediaPipe Hands 使用摄像头控制。
    -   **挥手 (移动)**: 旋转牌阵（包含惯性物理效果）。
    -   **捏合 (拇指+食指)**: 将选中的牌抽取至屏幕中心。
    -   **张开手掌**: 将牌放回牌阵中。
-   **视觉效果**: 深邃的氛围灯光与流畅的 GSAP 动画。

## 安装与运行

1.  **安装依赖**
    ```bash
    npm install
    ```

2.  **启动开发服务器**
    ```bash
    npm run dev
    ```

3.  **使用说明**
    -   根据浏览器提示允许摄像头权限。
    -   稍微站远一点，确保手部完整出现在画面中。
    -   **左右移动手势**: 旋转牌轮。
    -   **指向**: 悬停在卡片上（卡片会放大）。
    -   **捏合 (拇指+食指)**: 抽出卡片。
    -   **张开手掌**: 放回卡片。

## 技术栈
-   **Three.js**: 3D 渲染引擎。
-   **MediaPipe**: 手部追踪 AI。
-   **GSAP**: 动画库。
-   **Vite**: 构建工具。
